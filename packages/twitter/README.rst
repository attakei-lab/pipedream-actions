==================
X(Twitter) actions
==================

Send tweet
==========

:id: twitter_send-tweet

Send tweet by X API v2.

Props
-----

credentialText
  JSON string included credentials to access X API.
  Parsed object has these attributes:

  * api_key
  * api_key_secret
  * access_token
  * access_token_secret

bodyText
  Main message text of post.
